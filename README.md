# 2048
The once popular game 2048, now using Typescript, HTML5 Technology

# how to play

# step 1
npm install
# step 2 
npm run dev


#you can choise a difficult to start  in the begining
```typescript   
enum Difficult {
    /** has 8 * 8 cells */
    Easy,
    /** has 4 * 4 cells */
    Normal,
    /** has 16 * 16 cells */
    Hard,
    /** has 32 * 32 cells */
    Expert,
    /** has 64 * 64 cells */
    Boss
}
```
